# library-manager
这是我数据库引论的期中project
实现了一个图书管理系统
其中 `create_table2.sql` 可以用来建立数据库，运行 `crawlers/spider.py` 可以向数据库中添加大量豆瓣图书内容。

## 依赖
- Node.js
- npm
- MySQL

## 使用方法
切换到源代码所在目录
```bash
npm update
node app.js
```
